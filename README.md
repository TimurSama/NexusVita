## Nexus Vita - Экосистема здоровья, тренировок и тренеров

## 🎯 Описание

Nexus Vita — это модульная децентрализованная экосистема для пользователей, тренеров, врачей и фитнес-клубов. Приложение объединяет в себе:

- **Персональные страницы тренеров (мини-сайты)** - каждый тренер создаёт свой уникальный мини-сайт с кастомным дизайном
- **Интерактивный аватар тела** - отслеживание прогресса, целей и проблемных зон на схематичной модели тела
- **Система тем и модулей** - модульная система оформления с возможностью создания собственных тем
- **Токеномика и внутренняя экономика** - система токенов за активности, использование в магазине
- **Децентрализация** - блокчейн инфраструктура, токены, DAO управление
- **Человекоцентричный подход** - комплексное понимание здоровья во всех аспектах
- **Интеграция с Telegram** - синхронизация чатов, мини-приложение, уведомления
- **Медицинский модуль** - хранение анализов, рекомендаций врачей
- **Образовательный центр** - библиотека знаний, статьи, видео, гайды

## 🎨 Дизайн

Приложение выполнено в уникальном стиле **кибер-ренессанса**:
- Эффекты карандашных набросков (стиль Леонардо да Винчи)
- Material Expressive 3 анимации
- Пергаментные текстуры и чернильные линии
- Современный UX с плавными переходами

## 🚀 Быстрый старт

### Требования
- Node.js 18+
- PostgreSQL 14+
- npm или yarn

### Установка

```bash
# Клонируйте репозиторий
git clone <repository-url>
cd nexus-vita

# Установите зависимости
npm run install:all

# Настройте базу данных
cd backend
cp .env.example .env
# Отредактируйте DATABASE_URL в .env

# Запустите миграции
npx prisma migrate dev
npx prisma generate

# Запустите проект
cd ..
npm run dev
```

### Доступ
- Фронтенд: http://localhost:3000
- Бэкенд API: http://localhost:3001
- Демо-режим: http://localhost:3000/demo

## 🎭 Демо-режим

Приложение включает полнофункциональный демо-режим с презентациями для каждой роли:

- **Гость** - просмотр тренеров, контента, тем
- **Пользователь** - главная страница, аватар тела, тренировки, магазин, челленджи
- **Тренер** - страница тренера, управление контентом, программы, клиенты, календарь, магазин
- **Менеджер локации** - управление локацией, абонементы, контроль доступа, партнёрства
- **Администратор** - модерация, каталог тем, токеномика, аналитика

**Доступ:** http://localhost:3000/demo или через кнопку переключения ролей в хэдере
