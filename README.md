# Nexus Vita - Децентрализованная экосистема здоровья

Web3 платформа для фитнеса, здоровья, психологии и образования с DAO управлением, NVT токеном, NFT достижениями и Move-to-Earn механикой.

## 🚀 Быстрый старт (для демо-презентации)

### Локальная разработка (рекомендуется)

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```

Приложение будет доступно:
- Frontend: http://localhost:3000
- API: http://localhost:5000 (если запущен)

### Docker (опционально, только для production)

Docker нужен только для production-развертывания. Для локальной разработки и демо используйте `npm run dev`.

```bash
# Запуск через Docker (только для production)
docker-compose --profile production up -d
```

## 📦 Модули платформы

- 💪 **Фитнес & Тренировки** - персональные программы, Move-to-Earn
- 🥗 **Питание & Нутрициология** - планы питания, рецепты
- 🏥 **Здоровье & Медицина** - телемедицина, анализы
- 🧠 **Психология** - медитации, консультации
- 📊 **Привычки** - трекинг, gamification
- 📚 **Образование** - курсы, сертификации
- 🗳️ **DAO** - децентрализованное управление
- 🪙 **Web3** - NVT токен, NFT, стейкинг
