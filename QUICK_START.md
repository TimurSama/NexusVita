# Быстрый старт Nexus Vita

## 🚀 Запуск локально

```bash
# 1. Установка зависимостей
npm run install:all

# 2. Настройка БД (опционально для демо)
cd backend
cp .env.example .env
# Отредактируйте DATABASE_URL в .env

# 3. Запуск
cd ..
npm run dev
```

Откройте http://localhost:3000

## 🎭 Демо-режим

1. Откройте http://localhost:3000/demo
2. Или нажмите кнопку переключения ролей в хэдере
3. Изучите все функции без регистрации!
