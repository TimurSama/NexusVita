## Ключевые компоненты

### Фронтенд

#### Страницы
- **Главная (`/`)** - интерактивный аватар тела и виджеты
- **Демо (`/demo`)** - демо-режим со всеми ролями и функциями
- **Вход (`/login`)** - страница входа
- **Регистрация (`/register`)** - страница регистрации
- **Магазин (`/shop`)** - магазин товаров и услуг
- **Тренеры (`/trainers/[slug]`)** - страницы тренеров (мини-сайты)

#### Компоненты
- **BodyAvatar** - интерактивный аватар тела с отметкой целей и проблемных зон
- **TrainerPage** - полноценная страница тренера с модулями
- **ThemeProvider** - система тем оформления
- **DemoBanner** - баннер демо-режима

#### Контексты
- **AuthContext** - управление аутентификацией
- **DemoContext** - управление демо-режимом

### Бэкенд

#### API Роуты
Все роуты находятся под `/api/*`:
- `/api/auth` - регистрация, вход, получение текущего пользователя
- `/api/users` - профили пользователей
- `/api/trainers` - тренеры и их страницы
- `/api/posts` - публикации
- `/api/workouts` - программы и записи тренировок
- `/api/nutrition` - планы питания
- `/api/locations` - локации (залы, центры)
- `/api/shop` - товары и заказы
- `/api/chat` - чаты и сообщения
- `/api/events` - события (тренировки, вебинары)
- `/api/challenges` - челленджи
- `/api/tokens` - токены и транзакции
- `/api/themes` - темы оформления

#### База данных
Схема Prisma включает все необходимые модели:
- Пользователи и роли
- Тренеры и их страницы
- Темы и модули
- Контент (посты, комментарии)
- Тренировки и питание
- Локации и абонементы
- Магазин и заказы
- Токены
- Чат и сообщения
- События и челленджи
- Достижения

## Демо-режим

Демо-режим доступен по адресу `/demo` и включает:

### Роли
1. **Гость** - просмотр публичного контента
2. **Пользователь** - полный функционал клиента
3. **Тренер** - управление страницей и контентом
4. **Менеджер локации** - управление залом
5. **Администратор** - полный доступ

### Функции для каждой роли
- Главная страница
- Аватар тела
- Страница тренера
- Магазин
- Календарь
- Чат
- И другие...

## Система тем

### Базовые темы
1. **По умолчанию** - пергамент, чернильные наброски
2. **Дмитрий** - лес, грибы, Dune-атмосфера
3. **Олег** - киберпанк-неон

### Применение
Темы применяются через CSS переменные и атрибут `data-theme` на `<html>`.

## Следующие шаги

1. **Настройка БД**: Создайте PostgreSQL БД и настройте `.env`
2. **Миграции**: Запустите `npx prisma migrate dev`
3. **Запуск**: `npm run dev`
4. **Изучение**: Откройте `/demo` для изучения всех функций
