# Система тем Nexus Vita

## Концепция

Nexus Vita использует модульную систему тем, которая позволяет:
- Тренерам создавать уникальные мини-сайты с собственным оформлением
- Пользователям выбирать тему для всего приложения
- Дизайнерам создавать и публиковать темы в каталоге

## Структура темы

Тема состоит из:
- **Метаданные**: название, описание, автор
- **Конфигурация**: цвета, шрифты, текстуры, анимации

## Базовые темы

### 1. По умолчанию (Пергамент)
- Стиль: кибер-ренессанс, пергамент, чернильные наброски
- Цвета: сепия, крем, графит
- Акценты: бирюзовый, мягкий красный

### 2. Лес, грибы, Dune (Дмитрий)
- Стиль: природная эстетика, мягкие тона
- Цвета: бежево-зелёные, мхи, древесина
- Акценты: зелёный, коричневый

### 3. Киберпанк-неон (Олег)
- Стиль: sci-fi, неон, тёмный фон
- Цвета: фиолетовый, синий, антрацит
- Акценты: голубой неон, розовый, кислотный лайм

## Применение темы

Темы применяются через CSS переменные:

```css
[data-theme="dmitry"] {
  --theme-bg-primary: #f5f0e8;
  --theme-accent: #6b8e5a;
  /* ... */
}
```

## Создание темы

1. Создать конфигурацию темы в БД
2. Определить CSS переменные
3. Добавить текстуры и ресурсы
4. Опубликовать в каталоге

## Модули страниц тренеров

Тренеры могут выбирать модули для своей страницы:
- Профильный блок
- Лента публикаций
- Календарь
- Магазин
- Достижения
- Образовательный модуль
- И другие

Каждый модуль может иметь свою конфигурацию в рамках темы.
