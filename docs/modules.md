# Модули страниц тренеров

## Концепция модульности

Каждая страница тренера состоит из модулей, которые можно:
- Включать/выключать
- Настраивать порядок отображения
- Кастомизировать конфигурацию

## Доступные модули

### 1. Профильный блок
**Тип:** `profile`

Отображает:
- Аватар тренера
- Имя и специализации
- Краткое описание
- Кнопки взаимодействия

**Конфигурация:**
- Показывать/скрывать статистику
- Размер аватара
- Расположение элементов

### 2. Лента публикаций
**Тип:** `feed`

Отображает:
- Публикации тренера
- Фильтры по категориям
- Взаимодействие (лайки, комментарии)

**Конфигурация:**
- Количество постов на странице
- Типы контента для отображения
- Сортировка

### 3. Календарь и расписание
**Тип:** `calendar`

Отображает:
- Календарь на месяц
- Отмеченные дни тренировок
- Расписание событий

**Конфигурация:**
- Показывать только доступные слоты
- Типы событий для отображения

### 4. Магазин тренера
**Тип:** `shop`

Отображает:
- Услуги тренера
- Программы тренировок
- Цифровые продукты
- Цены в валюте и токенах

**Конфигурация:**
- Категории товаров
- Сортировка и фильтры

### 5. Достижения и награды
**Тип:** `achievements`

Отображает:
- Бейджи и достижения тренера
- Рейтинги
- Статистику

**Конфигурация:**
- Какие достижения показывать
- Формат отображения

### 6. Образовательный модуль
**Тип:** `education`

Отображает:
- Статьи и гайды
- Видеоуроки
- Подборки литературы
- Платные курсы

**Конфигурация:**
- Категории контента
- Фильтры по уровню

### 7. Локации тренера
**Тип:** `locations`

Отображает:
- Где работает тренер
- Расписание в каждой локации
- Партнёрские скидки

**Конфигурация:**
- Показывать карту
- Фильтры по типу локации

### 8. Медицинский модуль
**Тип:** `medical`

Отображает:
- Медицинские рекомендации
- Формы сбора анамнеза
- Интеграция с анализами

**Конфигурация:**
- Доступ только для авторизованных клиентов
- Уровень детализации

### 9. Социальный/челлендж модуль
**Тип:** `challenges`

Отображает:
- Активные челленджи тренера
- Групповые программы
- Рейтинги участников

**Конфигурация:**
- Типы челленджей
- Показывать лидеров

## Настройка модулей

Тренер может:
1. Выбрать активные модули
2. Настроить порядок отображения
3. Задать конфигурацию каждого модуля
4. Сохранить изменения

Конфигурация хранится в `TrainerPageModule.config` как JSON.

## Расширяемость

Новые модули можно добавлять без изменения ядра системы:
1. Создать компонент модуля
2. Зарегистрировать в системе
3. Добавить конфигурацию в БД
4. Тренеры смогут использовать новый модуль
