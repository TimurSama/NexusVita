# Интеграция с Telegram

## Концепция

Nexus Vita интегрируется с Telegram для:
- Синхронизации чатов
- Мини-приложения в Telegram
- Уведомлений

## Синхронизация чатов

### Принцип работы

1. Пользователь начинает диалог с тренером в приложении
2. Система создаёт или находит Telegram-чат
3. Сообщения синхронизируются в обе стороны:
   - Сообщение в приложении → появляется в Telegram
   - Сообщение в Telegram → появляется в приложении

### Реализация

- Использование Telegram Bot API
- Webhook для получения сообщений из Telegram
- Отправка сообщений через Bot API
- Связь чатов через `telegramChatId` в модели `Chat`

## Мини-приложение в Telegram

### Функционал

Облегчённая версия приложения:
- Просмотр страниц тренеров
- Лента публикаций
- Бронирование тренировок
- Чат
- Базовые функции магазина

### Технические детали

- Использование Telegram Web App API
- Синхронизация данных с основным приложением
- Адаптивный интерфейс для мобильных устройств

## Уведомления

### Типы уведомлений

1. **Тренировки**: напоминания о предстоящих тренировках
2. **Сообщения**: новые сообщения от тренеров
3. **Челленджи**: обновления прогресса
4. **Токены**: получение токенов за активности
5. **Заказы**: статус заказов

### Отправка

- Через Telegram Bot API
- Настройки уведомлений в профиле пользователя
- Возможность отключения типов уведомлений

## Настройка

### Переменные окружения

```env
TELEGRAM_BOT_TOKEN=your-bot-token
TELEGRAM_WEBHOOK_URL=https://your-domain.com/api/telegram/webhook
```

### Регистрация бота

1. Создать бота через @BotFather
2. Получить токен
3. Настроить webhook
4. Настроить мини-приложение (если нужно)

## Безопасность

- Проверка подписи запросов от Telegram
- Валидация данных
- Ограничение доступа к функциям бота
