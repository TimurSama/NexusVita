# Релизный чек‑лист

## Перед релизом
- [ ] Обновить переменные окружения в prod.
- [ ] Запустить миграции Prisma.
- [ ] Проверить Stripe ключи и webhook.
- [ ] Проверить KYC провайдера (если включен).
- [ ] Прогнать `npm run lint`, `npm run type-check`, `npm run test`.
- [ ] Проверить критичные сценарии вручную:
  - регистрация/вход
  - просмотр медкарты
  - создание DAO‑предложения
  - оформление подписки

## Релиз
- [ ] Сборка: `npm run build`
- [ ] Деплой на production
- [ ] Проверить старт приложения

## После релиза
- [ ] Мониторинг ошибок (Sentry/логирование).
- [ ] Контроль платежей и webhooks.
